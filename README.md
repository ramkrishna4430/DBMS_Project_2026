# LogiChain 🚚

LogiChain is a dual-portal supply chain platform for customers and distributors. It highlights advanced DBMS concepts using custom SQL triggers, stored procedures, and functions to handle live inventory validation, automated sales logging, and secure order processing.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Backend:** Node.js, Express.js
* **Database:** Oracle Database

## ✨ Core Features
* **Role-Based Access:** Dedicated, secure portals for both Customers and Distributors.
* **DBMS Integration:** Heavy emphasis on database-level logic, including:
  * `Triggers` for stock validation and automated sales logging.
  * `Stored Procedures` for secure checkout and reporting.
  * `Functions` for dynamic calculations like loyalty discounts and star ratings.
* **Live Analytics:** Distributor dashboard with real-time stats and custom date-range reporting.
